# Modelo de datos

## Entregables

- `data-model.tex` y `data-model.pdf`: explicación técnica y de negocio.
- `domain-model-by-schema.puml`: visión conceptual.
- `relational-model-by-schema.puml`: modelo relacional completo.
- `views-and-projections.puml`: proyección de lectura.
- `schemas/`: diagramas individuales por schema.
- `physical-model-audit.md`: contraste de 40 tablas, 377 campos, 76 FK, constraints e índices.
- `schema-ownership-and-grants.md`: roles y matriz de privilegios PostgreSQL.
- `migration-strategy.md`: despliegue, lock, upgrade y rollback.
- `physical-model-gate.md`: evidencia y bloqueos de la fase 3.

El PDF fue compilado con `pdflatex`. Los archivos auxiliares de compilación no forman parte del entregable.
