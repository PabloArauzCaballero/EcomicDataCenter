# Common

Capacidades transversales sin reglas específicas del observatorio:

- autenticación y autorización externa;
- errores y traducción HTTP;
- observabilidad;
- validación Zod;
- utilidades de transacción y procesamiento por bloques.

No deben colocarse aquí reglas de datasets, observaciones, metodologías o calidad. Una abstracción solo se incorpora cuando reduce repetición semántica real y tiene un contrato pequeño.
