# Contratos estadísticos compartidos

Contiene contratos de dominio usados por más de un módulo sin pertenecer exclusivamente a ingestión o consulta.

Actualmente incluye el valor tipado de dimensión, compartido por:

- ingestión, para construir la identidad de una serie;
- consulta, para filtrar una serie por dimensiones.

No deben colocarse aquí services, repositories ni reglas específicas de un caso de uso.
