# Gobierno semántico y de metadatos

Administra catálogos, clasificaciones, estructuras, metodologías, datasets e indicadores. Las transiciones de versiones se validan en services y la publicación verifica dependencias requeridas.

Solo `METHODOLOGY_STEWARD` puede modificar este dominio. El módulo no administra identidad ni permisos persistentes; consume el actor validado por el proveedor JWT externo.
