# Prompts y reglas aplicadas

Esta carpeta conserva las instrucciones base usadas durante la elaboración del modelo.

- `index.md`: orden de lectura, diagramas y criterios de entrega.
- `programacionGeneral.md`: precisión, simplicidad, documentación y calidad.
- `programacionBackend.md`: referencia futura para la implementación, no aplicada como código en esta fase.

La carpeta `prompt` documenta el proceso de generación y no reemplaza a `docs`.
